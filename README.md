# Craving Crusher

A gamified craving management tool for kids ages 6-12. Part of the HealthQuest ecosystem.

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- React 18 + TypeScript + Vite
- Tailwind CSS + Framer Motion
- PWA with offline support
- Firebase Auth (Google Sign-In)
- Netlify hosting + Blobs
